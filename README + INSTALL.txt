Advanced Duty System - X1Studios (SheriffX1)

1) Drag and drop file 'X1S-AdvDutySystem' into resources.
2) 'ensure X1S-AdvDutySystem' in server.cfg
3) Restart your server.
4) Enjoy.

OnDuty - Command - "/dutymenu"
OffDuty - Command - "/offduty"
Supervisor Panel - Command - "/supervisormenu"

Make sure to add your DC server GUILDID, DC bots token, DC webhook, and DC role ids into the config

You may add as many departments as you want but each new department must a webhook and roles or it wont work.

Please join our Discord and open a ticket for any help
